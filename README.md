☀️ Inovatim: Akıllı Güneş Paneli Verimlilik Portalı
Bu proje, Inovatim Yarışması kapsamında geliştirilmiş, C# (ASP.NET) altyapısına sahip web tabanlı bir enerji takip ve analiz sistemidir. Yenilenebilir enerji kaynaklarının verimliliğini dijital ortamda izlemek ve raporlamak amacıyla tasarlanmıştır.

💡 Proje Amacı

Dijital İzleme: Güneş panellerinden gelen verilerin web üzerinden anlık takibi.


Verimlilik Analizi: C# algoritmaları kullanarak panellerin performans değerlerini hesaplama.

Erişilebilirlik: Web tabanlı arayüz sayesinde her yerden sistem kontrolü sağlama.

🛠️ Teknik Altyapı

Backend: C# (.NET) 

Frontend: HTML, CSS, JavaScript

Yarışma: Inovatim İnovasyon Yarışması Projesi
